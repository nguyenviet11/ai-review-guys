How does Prodigy AI work?

Prodigy AI operates through a combination of Python scripts, web-based UI, and active learning strategies.

  1. Go to your Python environment and install Prodigy with the provided package.
  2. Run the command-line interface using prodigy followed by a recipe (e.g., ner.manual) to start an annotation session.
  3. Load your data in supported formats (JSONL, CSV, or text files).
  4. Launch the web application at http://localhost:8080 to begin annotating.
  5. Select labels and annotate examples, accepting or rejecting based on the task requirements.
  6. Save your annotations and export them using the db-out command for further processing or model training.

Prodigy AI offers extensive customization and full control over data privacy, making it a strong choice for machine learning teams needing flexibility.
